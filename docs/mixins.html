Mixins

-v($property, $values, $webkit:true, $moz:true, $ms:true, $o:true)

This mixin will create a stack of css properties:values where the specified vendor prefixes are true.

Example:
@include -v(transform, rotate(45deg));

Outputs:
-webkit-transform: rotate(45deg);
-moz-transform: rotate(45deg);
-ms-transform: rotate(45deg);
-o-transform: rotate(45deg);
transform: rotate(45deg);
